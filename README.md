that emphasizes the industry's best practices for professional programming.

using a wide variety of languages, libraries, and frameworks, including but not limited to:
- Ruby
- Ruby on Rails
- JavaScript
- HTML/CSS
- Backbone.js
- JQuery
- AJAX
- SQL
